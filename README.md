# 🐋 OrcaForge

The master utility monorepo. heavily optimized helpers across the entire stack.

## Architecture
OrcaForge is a framework-driven (package-driven) monorepo powered by Bun workspaces and Biome. 

- `packages/core/` - Pure TypeScript/JavaScript logic.
- `packages/react/` - Strict, native React components and hooks.
- `packages/next/` - Next.js specific utilities (Server Actions, Hydration).
- `packages/expo/` - React Native mobile expansions.
- `packages/laravel/` - Pure PHP 8.2+ (Traits, Macros, Responders).
